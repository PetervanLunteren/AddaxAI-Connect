# Asia

Taxonomy mapping template for Asian camera trap projects.

| latin | common | examples |
|-------|--------|----------|
| cervidae | deer | sambar, chital, muntjac, sika deer, barasingha |
| bovidae | bovid | gaur, banteng, takin, serow, goral, nilgai, blackbuck |
| suidae | pig | wild boar, bearded pig, warty pig |
| tragulidae | chevrotain | mouse deer (lesser/greater chevrotain) |
| moschidae | musk deer | musk deer |
| tapiridae | tapir | Malayan tapir |
| rhinocerotidae | rhinoceros | Indian rhino, Sumatran rhino, Javan rhino |
| elephantidae | elephant | Asian elephant |
| canidae | canid | dhole, golden jackal, red fox, Tibetan fox |
| felidae | cat | tiger, leopard, snow leopard, clouded leopard, fishing cat, leopard cat |
| ursidae | bear | Asiatic black bear, sun bear, sloth bear, brown bear, giant panda |
| mustelidae | mustelid | honey badger, otters, yellow-throated marten, hog badger |
| viverridae | civet | common palm civet, masked palm civet, binturong |
| herpestidae | mongoose | Indian grey mongoose, crab-eating mongoose |
| hyaenidae | hyena | striped hyena (West/Central Asia) |
| manidae | pangolin | Sunda pangolin, Chinese pangolin, Indian pangolin |
| cercopithecidae | monkey | macaque, langur, snub-nosed monkey, proboscis monkey |
| hylobatidae | gibbon | all gibbons, siamang |

| rodentia | rodent | porcupine, bamboo rat, squirrels |

| aves | bird | all birds |

| mammalia | mammal | catch-all |
