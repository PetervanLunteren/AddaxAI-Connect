# Europe

Taxonomy mapping template for European camera trap projects.

| latin | common | examples |
|-------|--------|----------|
| cervidae | deer | roe deer, red deer, fallow deer, moose |
| bovidae | bovid | chamois, ibex, mouflon, European bison |
| suidae | pig | wild boar |
| canidae | canid | red fox, grey wolf, golden jackal, arctic fox |
| felidae | cat | Eurasian lynx, Iberian lynx, wildcat |
| ursidae | bear | brown bear |
| mustelidae | mustelid | European badger, pine marten, stoat, otter, polecat |
| mephitidae | skunk | not present, but harmless to include |
| erinaceidae | hedgehog | European hedgehog |
| sciuridae | squirrel | red squirrel, grey squirrel |
| castoridae | beaver | Eurasian beaver |
| rodentia | rodent | voles, dormice, rats (catch-all for smaller rodents) |
| lagomorpha | rabbit | European rabbit, brown hare, mountain hare |
| viverridae | civet | common genet (Iberia, southern France) |
| herpestidae | mongoose | Egyptian mongoose (Iberia) |
| aves | bird | all birds |
| reptilia | reptile | all reptiles |
| mammalia | mammal | catch-all |
