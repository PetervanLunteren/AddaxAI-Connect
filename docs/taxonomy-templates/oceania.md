# Oceania

Taxonomy mapping template for Australian, New Zealand, and Pacific Island camera trap projects.

Oceania is unique: marsupial families replace most placental mammal groups. Several invasive placental mammals are also common camera trap subjects.

## Native

| latin | common | examples |
|-------|--------|----------|
| macropodidae | macropod | kangaroo, wallaby, tree-kangaroo, pademelon |
| vombatidae | wombat | common wombat, southern hairy-nosed wombat |
| phascolarctidae | koala | koala |
| phalangeridae | possum | brushtail possum, cuscus |
| petauridae | glider | sugar glider, yellow-bellied glider |
| dasyuridae | dasyurid | quoll, Tasmanian devil, antechinus, numbat |
| peramelidae | bandicoot | bandicoot, bilby |
| ornithorhynchidae | platypus | platypus |
| tachyglossidae | echidna | short-beaked echidna |
| rodentia | rodent | native rats, native mice |
| aves | bird | all birds (cassowary, emu, kiwi, etc.) |
| reptilia | reptile | all reptiles (goannas, snakes, etc.) |
