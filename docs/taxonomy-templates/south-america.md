# South America

Taxonomy mapping template for South American camera trap projects.

| latin | common | examples |
|-------|--------|----------|
| cervidae | deer | brocket deer, marsh deer, pudu, huemul |
| tayassuidae | peccary | white-lipped peccary, collared peccary |
| camelidae | camelid | guanaco, vicuna |
| tapiridae | tapir | South American tapir, mountain tapir, Baird's tapir |
| canidae | canid | maned wolf, bush dog, crab-eating fox, culpeo |
| felidae | cat | jaguar, puma, ocelot, oncilla, margay, jaguarundi |
| ursidae | bear | Andean bear (spectacled bear) |
| mustelidae | mustelid | giant otter, neotropical otter, grison, tayra |
| procyonidae | raccoon | South American coati, crab-eating raccoon, kinkajou, olingo |
| mephitidae | skunk | hog-nosed skunk, striped skunk |
| didelphimorphia | opossum | common opossum, grey four-eyed opossum |
| myrmecophagidae | anteater | giant anteater, tamandua |
| dasypodidae | armadillo | nine-banded armadillo, giant armadillo, six-banded armadillo |
| cebidae | monkey | capuchin, squirrel monkey |
| atelidae | howler monkey | howler monkey, spider monkey, woolly monkey |
| callitrichidae | marmoset | marmoset, tamarin |
| rodentia | rodent | capybara, agouti, paca, porcupine |
| lagomorpha | rabbit | tapeti (forest rabbit) |
| primates | primate | catch-all for other primates |
| aves | bird | all birds |
| reptilia | reptile | all reptiles (caiman, tegu, etc.) |
| mammalia | mammal | catch-all |
